= SG6000、SG6100、SG6200 StorageGRIDアプライアンスおよびサービスアプライアンス用のBMCインターフェイス
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ベースボード管理コントローラ（BMC）のユーザーインターフェイスは、ハードウェアのステータス情報を提供し、アプライアンスの設定、監視、診断のために低レベルのハードウェアアクセスを可能にします。

BMCを使用して次のことができます：

* ハードウェアのインストールに関するトラブルシューティング
* SNMPを設定します
* アプライアンスの電源状態を確認する
* アプライアンスの識別用LEDを制御する


次の StorageGRID アプライアンスは BMC インターフェイスをサポートしています：

* SG100
* SG110
* SG120
* SG1000 からのアクセス
* SG1100
* SG1200
* SGF6024
* SGF6112を参照してください
* SGF6212
* SG6060 の設計
* SG6160
* SG6260


アプライアンスをインストールする場合は、以下の手順に従ってBMCを設定してください：

* link:../installconfig/changing-root-password-for-bmc-interface.html["BMCインターフェイスの管理者パスワードまたはrootパスワードの変更"]
* link:../installconfig/setting-ip-address-for-bmc-management-port.html["BMC 管理ポートの IP アドレスを設定します"]
* link:../installconfig/accessing-bmc-interface.html["BMC インターフェイスにアクセスします"]
* link:../installconfig/configuring-snmp-settings-for-bmc.html["SNMPを設定します"]
* link:../installconfig/setting-up-email-notifications-for-alerts.html["BMCアラートのEメール通知を設定する"]


アプライアンスをグリッドにインストール済みで、StorageGRIDソフトウェアが実行されている場合は、次の手順を実行します：

* link:../commonhardware/placing-appliance-into-maintenance-mode.html["アプライアンスをメンテナンスモードにします"] をクリックして、StorageGRIDアプライアンスインストーラにアクセスします。
* StorageGRID Appliance Installer を使用した BMC インターフェイスへのアクセス方法については、link:../installconfig/setting-ip-address-for-bmc-management-port.html["BMC 管理ポートの IP アドレスを設定します"]を参照してください。


BMCの設定後に使用できる機能の詳細については、link:../commonhardware/use-bmc.html["BMCインターフェイスの使用"]および https://kb.netapp.com/hybrid/StorageGRID/Platforms/How_to_use_StorageGRID_Appliance_BMC_with_vendor_supplied_user_guide["BMCユーザガイド"^]を参照してください。
