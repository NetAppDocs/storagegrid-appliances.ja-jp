====
. アプライアンスの設置場所を決定します。
+

CAUTION: E2860またはE4000コントローラシェルフ、あるいはオプションの拡張シェルフを取り付ける際は、機器が転倒するのを防ぐため、ラックまたはキャビネットの下から上に向かってハードウェアを取り付けてください。最も重い機器がキャビネットまたはラックの下部に配置されるようにするため、SG6000-CN、SG6100-CN、またはSG6200-CNコントローラをE2860またはE4000コントローラシェルフおよび拡張シェルフの上に取り付けてください。

+

NOTE: 設置作業に着手する前に、SG6000、SG6100、またはSG6200アプライアンスに同梱されているケーブル、あるいはご自身で用意するケーブルが、計画している配線レイアウトに対して十分な長さであることを確認してください。

. 必要なネットワークスイッチを取り付けます。を参照してください link:https://imt.netapp.com/matrix/#welcome["NetApp Interoperability Matrix Tool で確認できます"^] を参照してください。
